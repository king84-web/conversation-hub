🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Install Dependencies
   $ npm install

2. Create .env.local with API Keys
   NEXT_PUBLIC_FLUTTERWAVE_KEY=...
   PRODUCT_PRICE=5000
   MAILCHIMP_API_KEY=...
   MAILCHIMP_AUDIENCE_ID=...

3. Run Development Server
   $ npm run dev

4. Open http://localhost:3000 in Browser
